# Global Airbnb Performance Analysis 🏨✈️

## 📌 Project Overview
This project is an interactive data visualization dashboard that analyzes the global performance of Airbnb. It provides deep insights into market share, property pricing, listing growth trends, and user ratings across 10 major global cities.

## 📊 Key Highlights & Metrics
- **Total Listings:** 2.79 Lakhs+
- **Total Hosts:** 1.82 Lakhs+
- **Total Reviews:** 5.37 Million+
- **Cities Analyzed:** 10 (Paris, New York, Sydney, Rome, Rio de Janeiro, Istanbul, Mexico City, Bangkok, Cape Town, Hong Kong)

## 💡 Major Insights
1. **Historical Growth:** Airbnb reached its peak for new listings in 2015. There was a noticeable decline in 2016-2017 (likely due to local regulations), followed by another drop in 2019 due to the COVID-19 pandemic.
2. **Market Dominance:** Paris, New York City (NYC), and Sydney account for almost **50% of the total listings** and **59% of the total reviews**. Paris leads the market, potentially driven by the fact that hotel rooms are twice as expensive as Airbnb listings.
3. **Pricing by Property Type:** 
   - Hotel Rooms are the most expensive (~$800).
   - Private Rooms are the most affordable (~$462).
4. **User Ratings & Satisfaction:** 
   - **Top Rated:** Mexico City (94.8%) and Rio de Janeiro (94.6%) are the best-rated cities overall.
   - **Lowest Rated:** Hong Kong (89.7%) and Istanbul (91.1%).
   - **Weak Areas:** Across all locations, 'Cleanliness' and 'Value for money' generally scored the lowest in detailed reviews.

## 🛠️ Interactive Features
The dashboard includes an interactive bookmark feature that allows users to toggle between different levels of detail:
- **Overall Rating View:** Shows the average review percentage per city.
- **Detailed Rating View:** Displays a granular table breaking down ratings by specific metrics (Check-in, Cleanliness, Communication, Location, Value).
